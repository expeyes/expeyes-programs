Establecer la conexi√≥n

Para acceder al hardware EYES17, los m√≥dulos Python para eyes17 debe estar instalado Se supone que est√°n en un directorio denominado eyes17, que podr√≠a estar en su directorio personal o sobre ¬´ Python PATH ¬ª.

Todos los programas deben comenzar con las siguientes dos l√≠neas:

La variable p es el objeto de software que representa el hardware.

Las siguientes secciones explican las llamadas a la funci√≥n Python para acceder al hardware EYES17. Cada llamada a la funci√≥n se explica con un ejemplo de uso.

set_pv1(v), set_pv2(v)

Ajusta el voltaje de CC en PV1 y PV2. El intervalo para PV1 es de -5 a 5. El intervalo para PV2 est de -3,3 a 3,3.

print p.set_pv1(4)
print p.set_pv2(2.1)

Se muestra el valor asignado a la salida. Medir los voltajes con un volt√≠metro

get_voltage(entrada)

Devuelve el voltaje de la entrada especificada.

print p.get_voltage('A1')
print p.get_voltage('A2')
print p.get_voltage('A3')
print p.get_voltage('MIC')
print p.get_voltage('SEN')

Conectar PV1 en A1, y utilizar set_pv1() y get_voltage('A1') juntos. Esta funci√≥n ajusta la plantilla de entrada por prueba y error, dependiendo del valor de la se√±al de entrada..

get_voltage_time(entrada)

Devuelve una tupla, que contiene el sello de fecha de la computadora y el voltaje en la entrada especificada..

Devuelve el valor de la resistencia conectada en SEN, que deber√≠a ser entre 100Œ© y 100kŒ© para una precisi√≥n razonable.

set_sine(frecuencia)

Genera la se√±al sinusoidal de la frecuencia solicitada en WG (intervalo de 5Hz a 5000Hz). Todos los valores intermedios no son posibles, la funci√≥n devuelve el valor realmente adoptado.

set_sine_amp(amplitud)

La amplitud se puede establecer en tres valores de voltaje preestablecidos en las cresta (0 para tener 80mV, 1 para tener 1V, 3 para tener 3V).

Ajusta la amplitud en 3V cresta.

set_sqr1(frecuencia)

Ajusta la frecuencia de la salida SQ1 (intervalo de 4Hz en 1MHz). Todos los valores intermedios no son posibles, la funci√≥n devuelve el valor realmente adoptado.

print p.set_sqr1(15030)

15030.53

set_sqr1_slow(frecuencia)

Ajusta la frecuencia de la salida SQ1 (intervalo de 0,‚ÄÖ1Hz a 1MHz). Todos los valores intermedios no son posibles, la funci√≥n devuelve el valor realmente adoptado. La resoluci√≥n est√° bien, pero WG est√° deshabilitado cuando se ordena SQ1 de esta manera.

set_sqr2(frecuencia)

Similar a set_sqr1() pero SQ2 no est√° disponible al mismo tiempo que WG, s√≥lo se puede usar una de las dos salidas a la vez.

Mide la frecuencia de una se√±al cuadrada en la entrada, IN2 o SEN. Conectar SQ1 en IN2 e iniciar el c√≥digo:

r2ftime(entrada1, entrada2)

Mide el intervalo de tiempo entre un flanco ascendente en la entrada1 hasta un flanco descendente en la entrada2, las entradas pueden ser las mismas. Se puede usar para probar una onda cuadrada.

Conectar SQ1 en IN2 y lanzar el c√≥digo :

print p.r2ftime('IN2', 'IN2')

0.0003

La se√±al cuadrada en 1kHz y permanece en el estado ALTO durante 0,‚ÄÖ3ms.

multi_r2rtime(entrada, nCiclos)

Mide el intervalo de tiempo entre dos bordes ascendentes de la entrada entrada. La duraci√≥n entre dos bordes ascendentes es un ciclo. Se puede especificar el n√∫mero de ciclos a medir, el valor predeterminado es 1. Los valores permitidos son 1,2,4,8,12,16,32 y 48. Esto se puede probar con una se√±al cuadrada.

Conectar SQ1 en IN2 y lanzar el c√≥digo :

select_range(canal, calibre)

Los calibres para A1 y A2 se pueden ajustar desde ¬±0.5V hasta ¬±16V, utilizando amplificadores de ganancia programables.

p.select_range('A1', 8)         # 8 volt maximum

mostrar√° dos tablas de fechas y voltajes.

set_wave(frecuencia, tipoDeSenal)

Si el par√°metro tipoDeSenal no est√° especificado, esto genera una signal se√±al utilizando la tabla de ondas existente. Si tipoDeSenal est√° especificada 'sine' ou 'tria', la tabla de onda correspondiente est√° cargada.

plot(x,y)
p.set_wave(100, 'tria')  # tabla de onda triangular y 100 Hz
x,y = p.capture1('A1', 500,50)
plot(x,y)
show()

load_equation(funcion, intervalo)

Crea la tabla de ondas usando la ecuaci√≥n. Conecte WG a A1 e inicia el c√≥digo siguiente. El resultado se muestra a continuaci√≥n.

from matplotlib import pyplot as plt

def f1(x):
    return sin(x) + sin(3*x)/3

p.load_equation(f1, [-pi,pi])
p.set_wave(400)
x,y = p.capture1('A1', 500,10)
plot(x,y)
show()

load_table(tabla)

Tambi√©n puede cargar la tabla de ondas con una matriz de 512 elementos. Conecte WG a A1 e inicie el c√≥digo a continuaci√≥n. Despu√©s de la operaci√≥n de valor absoluto, la tabla comienza con 256, luego baja a 0 y finalmente vuelve a 255, que dibuja una se√±al triangular. El resultado debido a esta tabla se muestra arriba.

from matplotlib import pyplot as plt
x = arange(-256, 256)
x = abs(x)
p.load_table(x)
p.set_wave(400)
x,y = p.capture1('A1', 500, 10)
plot(x,y)
show()
